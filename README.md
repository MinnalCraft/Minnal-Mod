# MINNAL MOD BOT

**Simple Node.js Based Modration Bot**

****
*Dependencies* : `discord.js@12.5.3` , `fs` , `ms` , `dotenv` , `express`

****
## SETUP 

**SETP 1** :- Install dependencies
IN TERMINAL\
`npm i discord.js@12.5.3`\
`npm i fs`\
`npm i ms`\
`npm i dontenv`\
`npm i express`

**STEP 2** :- Run the code
IN TERMINAL
Use `node index.js` To run the code

**AND NOW YOUR BOT IS RUNNING**
